# Petrodactal Monitor

A modern Pterodactyl Game Server Status Dashboard with real-time monitoring, beautiful UI, and one-click updates.

## Release Notes

### v1.0.0 (May 2026)
- Modern dark UI with live search, filtering, and copy buttons
- Custom Django Admin theme
- Clan banner support
- One-click update system with version checking
- Docker support
- Prometheus metrics + Grafana ready
- Game query plugins (Minecraft)
- Comprehensive tests and improved poller
- Full README with troubleshooting and post-install checklist

### v0.1 (Initial)
- Basic Pterodactyl integration and polling
