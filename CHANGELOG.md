# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.2.0] - 2026-05-20
### Added
- Full Setup Wizard with step-by-step guided configuration
- Advanced Discord rich embeds, high CPU/RAM warnings, daily summaries, and per-clan webhooks
- Dedicated Restart Services admin page with confirmation dialogs
- Poller status indicator in admin header
- Render one-click deploy (`render.yaml`)
- GitHub Actions CI workflow (linting + tests)
- Enhanced embed buttons and iframe support

### Improved
- Admin configuration options and SiteConfig model
- Error handling and poller resilience
- Demo Mode
- Documentation (full README + troubleshooting)

### Fixed
- Various minor bugs and missing files

## [1.1.0] - 2026-05-20
### Added
- Server detail modal and improved JavaScript (live search, filters, copy)
- Docker support
- Custom dark admin theme
- Clan banner support
- One-click update system with version checking

## [1.0.0] - 2026-05-20
### Added
- Initial Pterodactyl monitoring dashboard
- Basic polling and public pages
- Custom admin interface
